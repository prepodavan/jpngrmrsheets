## Verbs

### Present. て-form 接続形 (せつぞくけい, Setsuzokukei)

present

|   type    | infinitive     | connective 　　|
|-----------|----------------|----------------|
|   ichidan | る              | て         　　|
|     godan | う　/　つ　/　る　|　って       　　|
|     godan | ぬ　/　ふ　/　む　|　んで       　　|
|     godan | く　/　ぐ      　|　いて / いで 　　|
|     godan | す　            |　して       　　|
| irregular | する　          |　して       　　|
| irregular | 来る　(くる)    　|　来て (きて)　　|
| irregular | 行く　(いく)    　|　行って (いって)|

### Past. た-form 過去形 (かこけい, Kakokei)

past. same as て-form　but with た

## い-adjectives

### Present

い -> くて. いい (good)　-> よくて.

### Past

い -> かった.

## noun + copula (だ/です-clauses)

### Present

だ/です -> で

### Past

だ/です -> だった
です -> でした
